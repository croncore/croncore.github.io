Multi-Site IT Modernization & Cybersecurity for a U.S. Nonprofit

U.S. NONPROFIT · VIA PRIME (NDA)
5 Sites · 100% Endpoints Modernized

Cloud migration, Microsoft 365 modernization, network refresh, and cybersecurity hardening across five facilities — delivered remotely as a technical execution partner under NDA.

Confidentiality note. This case study describes work performed by Croncore as a remote technical delivery partner to a prime contractor under a confidentiality agreement. The end client, prime contractor, and any specific figures protected by NDA have been anonymized or omitted. Capabilities, technologies, frameworks, and outcomes described below accurately reflect Croncore's scope of responsibility on the engagement.

The Challenge

Our prime contracting partner was awarded a comprehensive IT modernization program for a long-established nonprofit operating across five buildings in a major U.S. metropolitan area. The client supports vulnerable communities, which raised the bar on data sensitivity, compliance, and uptime.

The existing environment carried a familiar mix of risks for a maturing nonprofit:

  • A legacy Azure server still anchoring file and identity management.
  • Microsoft 365 in place but without modern security baselines, DLP, conditional access, or SSO.
  • Aging switches and wireless access points across five physical sites, with no VLAN segmentation between staff, servers, and guest networks.
  • No tested backup or disaster recovery process for Microsoft 365 data.
  • Windows 10 workstations approaching end-of-life and incompatible with Windows 11.
  • Limited phishing protection, no enterprise password management, and inconsistent MFA enforcement.

Croncore was brought in by the prime as the technical execution partner to design, build, and stand up the cloud, identity, security, and managed-services layers of the program — working remotely under NDA, with on-site physical work handled by the prime's field teams.

Croncore's Role

Croncore did not contract directly with the end client. We operated as a remote technical delivery partner to the prime contractor, scoped, supervised, and presented to the end client by the prime under a confidentiality agreement. Our scope covered the engineering and managed-services workstreams where we are strongest: cloud identity, Microsoft 365 security, endpoint management, backup, and ongoing monitoring.

Within that arrangement, Croncore was responsible for:

  • Architecture and engineering for the cloud and identity migration.
  • Microsoft 365 security hardening and policy configuration.
  • Endpoint management design and rollout via Intune.
  • Backup, disaster recovery, and recovery testing for Microsoft 365 data.
  • Cybersecurity tooling deployment (phishing protection, password management, MFA).
  • Documentation, runbooks, and knowledge transfer back to the prime.
  • Ongoing remote monitoring, helpdesk triage support, and monthly performance reporting.

What We Delivered

01
Cloud Identity Migration
Decommissioned the legacy Azure server and migrated identity and access management to Azure Active Directory / Entra ID. Re-platformed file and role management to cloud-native services. Implemented role-based access controls, secure authentication policies, and Endpoint Manager (Intune) integration.
02
Microsoft 365 Modernization
Re-baselined Microsoft 365 security configuration. Designed and deployed Data Loss Prevention (DLP) policies tailored to sensitive client data. Upgraded licensing to Business Premium / F3 tiers, enabled Single Sign-On (SSO) and conditional access, and authored BYOD policies limiting personal-device data synchronization.
03
Endpoint Security & Management
Built the Intune device management posture for the full workstation fleet. Enabled automated patching, standardized antivirus and managed EDR, and defined the upgrade path for Windows 10 endpoints not eligible for Windows 11.
04
Backup & Disaster Recovery
Implemented Dropsuite backups for the Microsoft 365 tenant (mail, OneDrive, SharePoint, Teams). Defined backup schedules and retention aligned to the client's risk profile. Documented recovery procedures and executed test restores to verify the DR posture.
05
Cybersecurity Hardening
Deployed Ironscales for phishing protection and ongoing simulation. Rolled out Bitwarden Enterprise password management for key staff. Enforced MFA, session lockout, and password policies organization-wide. Aligned the security baseline with NIST CSF, CIS Controls, Microsoft Secure Score, and CISA guidance.
06
Network Engineering Support
Designed VLAN segmentation separating servers, staff, and guest traffic across all five buildings. Specified managed PoE++ gigabit switches and WPA3-capable wireless access points. Designed redundant internet connectivity at each site and produced topology documentation handed back to the prime's on-site engineering team for physical installation.
07
Ongoing Managed Services
Proactive monitoring, backup verification, and network reporting. Helpdesk triage with SLA-aligned response targets. Monthly performance reporting covering security metrics, Secure Score trend, patch compliance, and phishing simulation results. Annual IT review, budgeting input, and planning support.

Why This Engagement Matters for Our Clients

This program is representative of the work Croncore takes on as a remote technical partner — whether engaged directly or through a prime. A few things travel from this engagement to every similar one we deliver:

  • Mission-sensitive environments. The end client served vulnerable communities. Every design decision considered the privacy of client records, financial data, and employee data.
  • Multi-site complexity, single security posture. Five buildings, one identity plane, one set of policies, one Secure Score to improve — without fragmenting the experience for end users.
  • Microsoft-native, vendor-lock-in-aware. We standardized on Microsoft 365, Entra ID, and Intune while keeping data ownership, portability, and administrative access cleanly with the client.
  • Operational rigor. Phased delivery, written sign-offs at each phase, contingency-aware budgeting, and monthly performance reporting from day one of steady-state.
  • Quiet partnership. We are comfortable operating behind a prime contractor under NDA. The client sees one face; the prime gets a senior engineering bench on demand.
